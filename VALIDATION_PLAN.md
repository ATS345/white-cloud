## 3. 性能验证

### 3.1 性能指标

| 指标 | 目标值 | 测量工具 |
|------|--------|----------|
| 首页加载时间 | < 3秒 | Lighthouse, WebPageTest |
| API 响应时间 | < 500ms | Postman, JMeter |
| 并发用户数 | > 500 | JMeter |
| 系统 CPU 使用率 | < 80% | Top, Grafana |
| 系统内存使用率 | < 80% | Top, Grafana |
| 数据库查询时间 | < 100ms | PostgreSQL EXPLAIN ANALYZE |

### 3.2 性能测试场景

1. **首页访问**
   - 模拟 1000 个并发用户访问首页
   - 测量平均加载时间

2. **游戏列表查询**
   - 模拟 500 个并发用户查询游戏列表
   - 测量平均响应时间

3. **游戏详情查询**
   - 模拟 500 个并发用户查询游戏详情
   - 测量平均响应时间

4. **购物车操作**
   - 模拟 200 个并发用户进行购物车操作
   - 测量平均响应时间

5. **支付流程**
   - 模拟 100 个并发用户进行支付操作
   - 测量平均响应时间
