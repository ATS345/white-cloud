## 4. 测试步骤

### 4.1 首页游戏展示测试步骤

1. 访问首页：http://localhost:3000
2. 观察首页游戏展示是否正常
3. 点击游戏卡片，观察是否跳转到游戏详情页
4. 点击"查看更多"按钮，观察是否跳转到游戏列表页

### 4.2 游戏列表页测试步骤

1. 访问游戏列表页：http://localhost:3000/games
2. 观察游戏列表是否正常显示
3. 使用筛选功能，观察游戏列表是否更新
4. 使用排序功能，观察游戏列表是否按指定顺序排序
5. 使用搜索功能，观察搜索结果是否符合预期
6. 点击游戏卡片，观察是否跳转到游戏详情页

### 4.3 游戏详情页测试步骤

1. 访问游戏详情页：http://localhost:3000/game/1
2. 观察游戏详情是否完整显示
3. 查看游戏截图，观察是否可以正常切换
4. 查看系统需求，观察是否显示完整
5. 点击"加入购物车"按钮，观察购物车徽标数量是否更新
6. 点击"立即购买"按钮，观察是否跳转到结算页

### 4.4 搜索功能测试步骤

1. 在首页搜索栏输入关键词
2. 点击搜索按钮
3. 观察是否跳转到搜索结果页
4. 观察搜索结果是否符合预期
5. 重复测试不同关键词和边界情况

### 4.5 购物车功能测试步骤

1. 访问购物车页面：http://localhost:3000/cart
2. 观察购物车是否为空
3. 访问游戏详情页，点击"加入购物车"按钮
4. 回到购物车页面，观察游戏是否成功加入
5. 修改游戏数量，观察总价是否重新计算
6. 移除游戏，观察游戏是否从购物车中移除
7. 清空购物车，观察购物车是否为空

## 5. 测试注意事项

1. 测试过程中要记录所有的错误和异常情况
2. 测试用例要覆盖各种边界情况
3. 测试过程中要注意性能表现，记录页面加载时间
4. 测试完成后要清理测试数据
5. 测试结果要及时反馈给开发团队
6. 测试过程中要注意用户体验，评估交互是否流畅
