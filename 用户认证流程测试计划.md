## 4. 测试步骤

### 4.1 注册功能测试步骤

1. 访问注册页面：http://localhost:3000/register
2. 填写注册表单，输入有效信息
3. 勾选用户协议
4. 点击注册按钮
5. 观察是否注册成功，是否跳转到首页
6. 验证用户信息是否正确显示

### 4.2 登录功能测试步骤

1. 访问登录页面：http://localhost:3000/login
2. 填写登录表单，输入有效信息
3. 点击登录按钮
4. 观察是否登录成功，是否跳转到首页
5. 验证用户信息是否正确显示

### 4.3 登出功能测试步骤

1. 登录成功后，点击右上角用户图标
2. 选择登出选项
3. 观察是否登出成功，是否跳转到首页
4. 验证是否显示登录和注册按钮

### 4.4 密码修改功能测试步骤

1. 登录成功后，进入个人中心
2. 点击密码修改选项
3. 填写密码修改表单，输入有效信息
4. 点击保存按钮
5. 观察是否修改成功，是否显示成功提示
6. 使用新密码重新登录，验证修改是否生效

## 5. 测试注意事项

1. 测试过程中要记录所有的错误和异常情况
2. 测试用例要覆盖各种边界情况
3. 测试过程中要注意安全性，确保敏感信息不会泄露
4. 测试完成后要清理测试数据
5. 测试结果要及时反馈给开发团队
